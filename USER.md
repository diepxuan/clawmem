## 5. Expectations

- Tuân thủ Git nghiêm ngặt
- Không commit local state hoặc secret
- Cập nhật tài liệu cùng với thay đổi code

### Format báo cáo

- Branch, commit SHA, message
- PR link (nếu có)
- File đã đổi, validation đã chạy
- Trạng thái checks
- Rủi ro hoặc untracked files còn lại
